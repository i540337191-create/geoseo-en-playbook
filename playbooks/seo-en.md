# 英文 SEO 执行手册

SEO EN 关注英文搜索可见度：页面可抓取、可索引、元信息、站点地图、目录资料页和基础外链。

## 核心规则

发布或提交任何外部内容之前，先确认英文目标页在线且可索引。

## 目标页检查

确认：

- URL 返回 `200`
- canonical 指向自身
- `<html lang="en">`
- title 和 description 是英文
- 页面没有 `noindex`
- sitemap 包含目标页

## Google Search Console

1. 打开正确的网站属性。
2. 检查英文目标 URL。
3. 如果页面新建、刚更新或结果过期，运行实时测试。
4. 页面可索引后，请求编入索引。
5. 检查 sitemap 是否已提交且状态成功。

## Bing Webmaster Tools

提交：

- 主页
- 英文目标页
- sitemap

记录每一项状态：`Done`、`Processing` 或被登录/验证阻塞。

## 目录与资料页

只有在平台要求公司官网时才使用主页。产品介绍、文章链接、发布链接和 SEO 外链优先使用英文目标页。

适合的平台：

- Medium
- StartupBase
- F6S
- Crunchbase
- Product Hunt，前提是产品素材已经准备好

遇到付费推广、重验证或需要夸大产品状态的平台，先跳过或标记为后续处理。

## 状态汇报

统一使用四类状态：

- Done
- Pending review
- Blocked by login / verification / payment
- Needs user action
