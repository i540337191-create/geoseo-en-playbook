# 每日执行清单

用于轻量执行一天的 GEO + SEO 工作。

## 1. 确认目标页

- [ ] 目标页仍然是本轮主推 URL。
- [ ] 页面返回 `200`。
- [ ] 页面可索引。
- [ ] sitemap 包含该页面。

## 2. Google Search Console

- [ ] 检查目标 URL。
- [ ] 如果页面新建或刚更新，请求编入索引。
- [ ] 检查 sitemap 状态。

## 3. Bing Webmaster Tools

- [ ] 必要时提交主页。
- [ ] 必要时提交英文目标页。
- [ ] 提交或检查 sitemap 状态。

## 4. GEO 发布

- [ ] 除非有明确理由，一天只选一个平台。
- [ ] 内容要符合平台语境。
- [ ] 品牌提及要自然。
- [ ] 只使用一个主链接。
- [ ] 发布后记录最终 URL。

## 5. 写入日志

- [ ] 平台
- [ ] 动作
- [ ] URL
- [ ] 日期
- [ ] 状态
- [ ] 后续事项
