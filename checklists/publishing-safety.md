# 发布安全清单

发布、提交或创建公开资料页之前，先检查这些事项。

- [ ] 用户已经明确授权发布。
- [ ] 没有选择付费升级、赞助展示或付款流程。
- [ ] 没有未经确认的产品上市、销量、客户或融资说法。
- [ ] 没有包含私密凭据、个人信息或内部备注。
- [ ] 删除品牌句子后，内容仍然有用。
- [ ] 目标链接正确。
- [ ] 内容格式符合平台语境。
- [ ] 发布后的最终 URL 已保存。

## 避免

- 同一篇内容复制到所有平台。
- 过度重复 exact-match SEO 关键词。
- 虚假夸赞。
- 虚构产品进展、客户、融资、合作或数据。
- 绕过验证、登录或平台规则。
